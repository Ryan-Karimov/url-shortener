# Вклад в проект

Спасибо за интерес к проекту! Мы рады любому вкладу.

## Как внести вклад

### 1. Сообщить об ошибке

- Проверьте, не существует ли уже такой issue
- Создайте новый issue с подробным описанием:
  - Шаги для воспроизведения
  - Ожидаемое поведение
  - Фактическое поведение
  - Версия Node.js, ОС

### 2. Предложить улучшение

- Создайте issue с тегом `enhancement`
- Опишите функционал и зачем он нужен

### 3. Отправить Pull Request

```bash
# Форкните репозиторий
# Клонируйте свой форк
git clone https://github.com/your-username/url-shortener.git

# Создайте ветку
git checkout -b feature/my-feature

# Внесите изменения и закоммитьте
git commit -m "feat: добавлена новая функция"

# Отправьте в свой форк
git push origin feature/my-feature

# Создайте Pull Request
```

## Стиль кода

- Используйте 2 пробела для отступов
- Точка с запятой в конце строк
- Одинарные кавычки для строк
- async/await вместо callbacks

## Коммиты

Используйте [Conventional Commits](https://www.conventionalcommits.org/):

```
feat: новая функция
fix: исправление бага
docs: обновление документации
refactor: рефакторинг кода
test: добавление тестов
```

## Вопросы

Если есть вопросы — создайте issue с тегом `question`.
